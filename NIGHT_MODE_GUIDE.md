## ğŸ”§ Technical Details

### CSS Variables
All colors are CSS variables that automatically switch with theme:
- `--background` - Background color
- `--foreground` - Text color
- `--primary` - Primary color (Green)
- `--accent` - Accent color

### Dark Mode Implementation
- Uses `dark:` Tailwind prefix
- Automatic based on system preference option
- Persistent in localStorage
- Smooth transitions

### Animations Used
- **Blob**: Organic floating shapes
- **Float**: Gentle vertical movement
- **Fade-in-up**: Entrance animation
- **Glow-pulse**: Pulsing glow effect
- **Shimmer**: Loading animation
- **Gradient-flow**: Flowing gradients
